# DAX Measures Overview

Folder ini berisi dokumentasi logika perhitungan (DAX measures) 
yang digunakan dalam dashboard Power BI untuk analisis data koperasi nasional.

Fokus utama penggunaan DAX:
- Konsistensi perhitungan indikator
- Analisis tren waktu (Year-over-Year)
- Penyajian KPI yang siap digunakan oleh user non-teknis

---

## Prinsip Perhitungan
- Measure digunakan untuk perhitungan dinamis sesuai filter
- Data telah melalui proses pembersihan pada tahap awal
- Perhitungan difokuskan pada indikator utama koperasi

---

## Daftar Measure Utama
- Total Koperasi Aktif
- Total Anggota
- Total Aset
- Total SHU
- Total Modal
- Total Volume Usaha
- Year-over-Year (YoY) Growth
